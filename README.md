# vue-music

A vue mobile music project

## 安装依赖

`npm install`

## 启动后台 API 服务

`npm run server`

## 启动一个具有热重载功能的开发环境

`npm start`

## 项目构建

`npm run build`

## 项目部署

`npm run deploy`

> 修改 deploy.sh 文件 中的部署命令
> scp -r server/\* 你的服务器用户名@你的服务器公网 IP:服务器项目存放目录
